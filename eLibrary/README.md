# eLibrary

    Browse books, and mark them if there is any favorite

### page *(function)*:
-   login *(JWT)*
    -   username
    -   e-mail
    -   FB account

-   register
-   book list *(show all the books)*
    -   order by price
    -   key word searching
    -   mark favorite books
-   favorite book list *(show book(s) that user likes best)*
-   re-edit profile
-   logout
